SDK’s¶

In this document, a x64 Debian Bookworm Docker image will be used as a sysroot for cross compilation.

  1. Install WSL and Rancher Desktop

  2. Pull the Debian Image, Build a container then install g++

    docker pull debian:bookworm
docker run -it --name=deb debian:bookworm
apt update; apt install g++; exit
docker export deb -o deb.tar

  3. Export the container as a .tar file and extract it as admin

docker export deb -o deb.tar
