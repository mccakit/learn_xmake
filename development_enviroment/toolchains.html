Toolchains¶

In this tutorial we are going to use LLVM/Clang for most tutorials, intel OneAPI will be used for SYCL, rustc will be used for rust.

LLVM/Clang¶

To install LLVM with all available backends, you must compile it from source. On Windows, this requires Visual Studio and additional dependencies.

  1. Install Visual Studio Build Tools

    1. Get the installer from here

    2. Enable Desktop Development with C++ and at the right panel enable ATL, MFC, CLI support and Modules

  2. Install additional dependencies

    1. Install scoop by following instructions in this link

    2. Install cmake, python, make, git with scoop install

  3. Build LLVM

    1. Clone the LLVM repository. I recommend using this PR branch as it enables Doxygen-style formatting on hover.

    Note

    Don’t forget to switch to clangd-doxygen-parser branch

    1. Open the x64 Native Tools Command Prompt and navigate to to your llvm repository

    2. Create build configuation file with cmake

    Note

    I recommend building clang, lld, clang-tools-extra, lldb, libcxx, libc and compiler-rt with exception handling, run-time type information and assertions enabled. For targets, I suggest building X86, ARM, RISC-V and WebAssembly.

    1. Build and install LLVM with ninja --build and `cmake --install

  4. Patch LLVM installation so C++ Modules work

    1. Clone this repo that contains patch files for llvm\share\libc++\v1\std just override the files

  5. Add the bin folder of your installation to your PATH
