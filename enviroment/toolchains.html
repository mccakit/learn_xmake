Toolchains¶

In this document, following toolchains will be used:

LLVM¶

To install LLVM with all available backends, you must compile it from source. On Windows, this requires Visual Studio and additional dependencies.

  1. Install Visual Studio Build Tools

    1. Get the installer from here

    2. Enable Desktop Development with C++ and at the right panel enable ATL, MFC, CLI support and Modules

  2. Install additional dependencies

    1. Install cmake, python, make and git with scoop install

  3. Build and Install LLVM

    1. Clone the LLVM repository. I recommend using this PR branch as it enables Doxygen-style formatting on hover.

      Note

      Don’t forget to switch to clangd-doxygen-parser branch

    2. Open the x64 Native Tools Command Prompt and navigate to to your llvm repository

    3. Create build configuation file with cmake

      Note

      I recommend building clang, lld, clang-tools-extra, lldb, libcxx, libc and compiler-rt with exception handling, run-time type information and assertions enabled. For targets, I suggest building X86, ARM and RISC-V

    4. Build and install LLVM with ninja --build and `cmake --install

  4. Patch LLVM installation so C++ Modules work

    1. Clone this repo that contains patch files for llvm\share\libc++\v1\std just override the files

  5. Add the bin folder of your installation to your PATH

EMCC¶

  1. Download emscripten using scoop

    scoop install main/emscripten

  2. Install and activate Emscripten

    emsdk install latest
emsdk activate latest --permanent

DPC++¶

  1. Download and run DPC++ installer from this link.

CUDA¶

  1. Download CUDA using scoop

    scoop install main/cuda

Rustc¶

  1. Download rustup using scoop

    scoop install main/rustup

Zig¶

  1. Download zig using scoop

    scoop install versions/zig-dev
