Editor¶

This document uses Zed Editor because it’s lightweight and works well with CLI tools. To configure a language, install the extension and specify the paths to your LSP and formatter binaries in the project settings.

  1. Install Zed Editor

    scoop install extras/zed

  2. Install extensions below via the Extensions tab in Zed

    Extensions¶

    Extension

    Description

    HTML

    HTML support

    Git Firefly

    Provides Git Syntax Highlighting

    Make

    Makefile syntax highlighting

    Lua

    Lua Support

    CSV

    CSV Language Support

    Python LSP

    Python LSP extension

    reST

    reST syntax support

  3. Install additional LSPs and formatters using tools

    Additional Tools¶

    Tool

    Installation Method

    Description

    lua-language-server

    Scoop

    LSP for Lua with fast parsing and full-featured language support

    rstfmt

    pip

    Formatter for reStructuredText documents

    jq

    Scoop

    Command-line JSON processor for filtering and transforming JSON data

    esbonio

    pip

    LSP server for reStructuredText and Sphinx-based documentation

    rust-analyzer

    cargo

    Modern LSP server for Rust with fast, IDE-friendly tooling support

    zls

    Scoop

    Zig Language Server providing autocompletion, diagnostics, and more

  4. Configure project settings to use your tools

{
    "lsp": {
        "clangd": {
            "binary": {
                "path": "C:/devtools/llvm/bin/clangd.exe",
                "arguments": ["--background-index"]
            }
        },
        "lua-language-server": {
            "binary": {
                "path": "C:/Users/xxx/scoop/apps/lua-language-server/current/bin/lua-language-server.exe"
            }
        },
        "esbonio": {
            "binary": {
                "path": "python",
                "arguments": ["-m", "esbonio"]
            }
        }
    },
    "languages": {
        "C++": {
            "formatter": {
                "external": {
                    "command": "C:/devtools/llvm/bin/clang-format.exe",
                    "arguments": ["--style=microsoft", "{buffer_path}"]
                }
            },
            "language_servers": ["clangd"]
        },
        "C": {
            "formatter": {
                "external": {
                    "command": "C:/devtools/llvm/bin/clang-format.exe",
                    "arguments": ["--style=microsoft", "{buffer_path}"]
                }
            },
            "language_servers": ["clangd"]
        },
        "Lua": {
            "language_servers": ["lua-language-server"]
        },
        "reST": {
            "language_servers": ["esbonio"],
            "formatter": {
                "external": {
                    "command": "rstfmt"
                }
            }
        },
        "HTML": {
            "language_servers": ["vscode-html-languageserver"]
        },
        "language_overrides": {
            "JSONC": {
                "formatter": {
                    "external": {
                        "command": "jq",
                        "arguments": ["--indent", "4", ".", "{buffer_path}"]
                    }
                }
            }
        }
    }
}
