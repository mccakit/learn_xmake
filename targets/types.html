Types¶

To set type of a target, call set_kind Here are the the all possible types for targets

Target Kinds¶

Value

Description

phony

Non-buildable alias target

binary

Executable program

static

Static library

shared

Shared (dynamic) library

object

Compiled object files only

headeronly

Headers only

Here is a basic example:

target("static_lib") do
    set_kind("static")
    add_files("src/static_lib/lib/*.cpp")
    add_headerfiles("src/static_lib/inc/*.hpp")
    add_includedirs("src/static_lib/inc", {public = true})
end
